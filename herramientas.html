Herramientas para aplicar Aprendizaje de computadoras

Existen muchas herramientas para programar IAs, citaré las que uso que, en opinion propia, me parecieron las mejores. Todas estas herramientas están disponibles para python y algunas tambien en C/C++.

Control de data-sets

En el uso del aprendizaje automático la necesidad de trabajar con matrices de datos es muy necesario. En este caso mi recomendación es:

NUMPY

Obtención:
  • Su dominio:www.numpy.org
  • Con python-pip: pip install numpy

PANDAS

Obtención:
  • Su dominio: www.pandas.pydata.org
  • Con python-pip: pip install pandas

Matemática

La Matemática no son el fuerte de muchos e incluso para los que sí, programar los cálculos y las fórmulas son perdida de tiempo. Existe una forma de evitarlo con una librería que saca esta carga.

SCIPY

Obtención:
  • Su dominio: www.scipy.org
  • Con python-pip: pip install scipy

Diagramas

Para ver la evolución y el funcionamiento de la IA la forma mas cómoda es diagramar. Hay una herramienta para su fácil y rápida aplicación.

MATPLOTLIB

Obtención:
  • Su dominio: www.matplotlib.org
  • Con python-pip: pip install matplotlib

Aprendizaje automático

Para la tarea principal hay varias herramientas, su elección depende del proyecto a llevar a cabo. Listaré algunas, pero hay que tener en cuenta de que son herramientas de generalización, pueden no siempre hacer exactamente lo buscado.

SCIKIT-LEARN

Obtención:
  • Su dominio: www.scikit-learn.org
  • Con python-pip: pip install skikit-learn

TENSORFLOW

Obtención:
  • Su dominio: www.tensorflow.org
  • Con python-pip: pip install tensorflow

KERAS

Obtención:
  • Su dominio: www.keras.io
  • Con python-pip: pip install keras
